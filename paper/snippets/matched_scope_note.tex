\paragraph{Matched-scope interpretation.}
The three empirical tracks should not be read as if they were all measured on the same denominator. Track~A is the primary result and uses the canonical 64-prompt suite with the first prompt excluded as warmup, yielding $N{=}63$ scored prompts across six categories. By contrast, the checked-in Track~B and Track~C artifacts use a legacy five-prompt expository subset with four scored prompts after warmup. We therefore treat Track~B as a scoped negative baseline and Track~C as boundary-condition evidence rather than as directly pooled estimates with Track~A's main table. When a direct cross-track comparison is needed, the conservative comparison is to slice Track~A to the same shared expository subset and report that matched-scope view separately, rather than mixing the canonical Track~A $N{=}63$ results with the smaller Track~B/Track~C denominator.
